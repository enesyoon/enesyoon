# Self-Health MVP - 30 Day Challenge

## Quick Start
1. Clone: `git clone https://github.com/enesyoon/self-health-mvp.git`
2. Env: Add Bohr API key to .env.local
3. Run: `docker-compose up --build`

## Tech Stack
- Frontend: Next.js 14 + Tailwind + Framer Motion
- Backend: FastAPI + Prisma (PostgreSQL)
- SDK: @bohrhealth/web-sdk
- Push: Firebase FCM
